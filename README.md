# 텍스트 분석 도구 (과제 1)

한글과 영문 텍스트 파일을 분석하여 다양한 통계 정보를 제공하는 웹 기반 도구입니다.

## 기능

- 📊 **다양한 통계 분석**
  - 총 문자 수 (공백 포함/제외)
  - 총 단어 수 (한글: 어절, 영문: 띄어쓰기 기준)
  - 총 문장 수 (., !, ? 기준)
  - 한글 문자 수
  - 영문 문자 수
  - 숫자 문자 수
  - 특수문자 수

- 🎨 **사용자 친화적 인터페이스**
  - 드래그 앤 드롭 파일 업로드
  - 반응형 디자인
  - 아름다운 그라디언트 UI

- 🛡️ **예외 처리**
  - 파일 미선택 검증
  - 빈 파일 검증
  - 파일 크기 제한 (10MB)
  - 파일 형식 검증 (.txt만 허용)
  - 인코딩 오류 처리 (UTF-8)

## 사용 방법

1. 웹 브라우저에서 `index.html` 파일을 엽니다
2. "파일 선택" 버튼을 클릭하거나 파일을 드래그하여 업로드합니다
3. `.txt` 형식의 텍스트 파일을 선택합니다
4. 자동으로 분석 결과가 테이블 형태로 표시됩니다

## 테스트

프로젝트에 포함된 `sample_text.txt` 파일을 사용하여 기능을 테스트할 수 있습니다.

## 기술 스택

- HTML5
- CSS3 (그라디언트, 애니메이션, 반응형 디자인)
- JavaScript (ES6+)
- FileReader API

## 파일 구조

```
.
├── index.html          # 메인 HTML 파일 (CSS/JS 포함)
├── sample_text.txt     # 테스트용 샘플 파일
├── README.md          # 프로젝트 설명
└── LICENSE.md         # 라이선스
```

## 특징

- 단일 HTML 파일로 구성되어 있어 별도의 설치나 빌드 과정이 필요 없습니다
- 모든 처리가 클라이언트 사이드에서 이루어져 빠르고 안전합니다
- 한글 정규표현식을 사용하여 정확한 한글 문자 인식이 가능합니다
- 다른 과제들과의 연결을 위한 네비게이션 링크 제공
